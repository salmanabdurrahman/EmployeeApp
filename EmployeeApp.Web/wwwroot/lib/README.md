# wwwroot/lib Directory

This directory contains third-party JavaScript libraries and CSS frameworks.

## Required Libraries

### ✅ Already Included (via dotnet new)
- **Bootstrap 5 **- UI framework (in `bootstrap/`)
- **jQuery **- JavaScript library (in `jquery/`)
- **jQuery Validation **- Form validation (in `jquery-validation/`)

### ⚠️ NOT Included - Must Download Separately

#### Kendo UI for jQuery (~210 MB)
**Location:  **`kendo-ui/` folder (currently empty or missing)

**Why not included?**
- Large file size (210 MB)
- Requires personal license key
- Each developer needs their own trial/license

**How to install:  **
1. See**[KENDO_SETUP.md](../../KENDO_SETUP.md)**for complete guide
2. Download from: https://www.telerik.com/download/kendo-ui
3. Copy `js/` and `styles/` folders here
4. Get license from: https://www.telerik.com/account/product-keys
5. Create `wwwroot/js/kendo-ui-license.js` with your key

**Required structure after installation:  **
```
lib/
├── kendo-ui/
│   ├── js/
│   │   ├── kendo.all.min.js      (required)
│   │   ├── kendo.all.min.js.map
│   │   └── ... (other files)
│   └── styles/
│       ├── default-main.css       (required)
│       └── ... (other themes)
```

**Verification:  **
```bash
# Check if installed correctly
ls kendo-ui/js/kendo.all.min.js
ls kendo-ui/styles/default-main.css

# Should show ~210M
du -sh kendo-ui/
```

## Git Ignore

Files in this directory are ignored by git (except this README):
- `wwwroot/lib/` is in `.gitignore`
- Developers must download Kendo UI separately
- Bootstrap/jQuery are regenerated by `dotnet new`

## Troubleshooting

**Problem:  **Web app shows "Kendo is not defined" error

**Solution:  **Kendo UI not installed. Follow [KENDO_SETUP.md](../../KENDO_SETUP.md)

---

**Problem:  **License warning in browser console

**Solution:  **Create `wwwroot/js/kendo-ui-license.js` with your license key

---

**Problem:  **Grid not showing / CSS broken

**Solution:  **Verify `kendo-ui/styles/default-main.css` exists
